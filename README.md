# VECTOR | Industrial Logistic Landing & CMS

Современный, высокопроизводительный лендинг для логистической компании, специализирующейся на печати и дистрибуции документов ЖКХ. Проект выполнен в глубоком индустриальном стиле с использованием передового фронтенд-стека и интегрированной системы управления контентом (CMS).

## 🚀 Стек технологий

- **React 18** — компонентная архитектура и хуки.
- **Tailwind CSS** — современная стилизация с кастомными расширениями.
- **Lucide React** — пак иконок в техническом стиле.
- **PWA Ready** — поддержка манифеста и Service Worker для оффлайн-доступа.
- **SEO Optimized** — настроенные мета-теги, JSON-LD Business Schema, robots.txt и sitemap.xml.
- **Local Storage CMS** — полное управление сайтом без необходимости в бэкенде.

## ✨ Ключевые особенности

- **Industrial Tech Aesthetic**: Глубокая цветовая палитра (Navy/Steel), шумные текстуры (noise filter), эффект сканирующих линий (CRT overlay) и шрифты Manrope / JetBrains Mono.
- **Динамическая CMS**: Редактирование текстов, структуры секций, цветовых тем, SEO-настроек и интеграций прямо в браузере.
- **Функциональные формы**: Валидация полей, индикация загрузки и опциональная интеграция с **Formspree** для реальных уведомлений.
- **Зона покрытия**: Интерактивная стилизованная карта регионов обслуживания (SVG).
- **Производительность**: Ленивая загрузка изображений, асинхронное декодирование и предзагрузка критических ресурсов.

## 🛠 Управление контентом (CMS)

Для входа в панель управления используйте комбинацию клавиш на сайте:
`Alt` + `Shift` + `C`

**Пароль по умолчанию:** `admin`

### Возможности CMS:
1. **Структура**: Изменение порядка секций и их видимости.
2. **Основные**: Редактирование юридических реквизитов и контактов.
3. **Оформление**: Настройка цветовых схем темной и светлой тем.
4. **География**: Редактирование списка городов и типов покрытия.
5. **Интеграции**: Поля для Formspree ID, Yandex Metrica, Google Analytics и Facebook Pixel.
6. **Экспорт/Импорт**: Резервное копирование всей конфигурации в JSON-файл.

## 📁 Структура проекта

- `src/components/` — отдельные React-компоненты (Hero, Features, ServiceArea, Stats и др.).
- `src/Landing.js` — главный контейнер лендинга с логикой тем и модальных окон.
- `src/CMS.js` — интерфейс и логика панели управления ("Industrial Panel").
- `src/App.js` — корневой компонент, управление состоянием, Error Boundary и аналитика.
- `public/` — манифест, Service Worker, логотипы и SEO файлы.

## 🏁 Запуск и деплой

1. Установите зависимости: `npm install`
2. Запустите проект: `npm start`
3. Сборка для продакшена: `npm run build`

### Рекомендации по деплою:
Проект полностью статичен и идеально подходит для:
- **Vercel** / **Netlify** — автоматический деплой из Git.
- **GitHub Pages** — бесплатный хостинг для статических сайтов.
- **Любой VPS** — через Nginx/Apache.

## 🔧 Настройка уведомлений
Для получения реальных заявок на почту:
1. Создайте форму на [Formspree.io](https://formspree.io).
2. Скопируйте ID формы.
3. Вставьте его в CMS во вкладке "Интеграции".
