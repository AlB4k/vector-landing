## Google Analytics 4

### Шаг 1: Создание проекта GA4

1. Откройте [Google Analytics](https://analytics.google.com)
2. Нажмите **"Создать"** (Create Account)
3. Заполните информацию:
   - **Название аккаунта**: `VECTOR`
   - **Название свойства**: `VECTOR Landing`
   - **Часовой пояс**: `Europe/Moscow`
   - **Валюта**: `RUB (₽)`
4. Нажмите **"Создать"**

### Шаг 2: Получение Measurement ID

1. В GA4 перейдите в **Администрация** → **Свойства**
2. В левом меню выберите **Потоки данных**
3. Нажмите на ваш веб-поток
4. Скопируйте **Measurement ID** (формат: `G-XXXXXXXXXX`)

### Шаг 3: Добавление в CMS

1. Откройте CMS вашего сайта (Alt+Shift+C на сайте)
2. Найдите поле **Google Analytics Measurement ID**
3. Вставьте полученный ID: `G-XXXXXXXXXX`
4. Нажмите **"Сохранить"**

**Альтернатива:** Если нет поля в CMS, отправьте ID разработчику.

### Шаг 4: Проверка установки

1. Откройте ваш сайт: https://vector-landing-roan.vercel.app
2. Нажмите F12 → вкладка **Network** → фильтр **Fetch/XHR**
3. Ищите запросы к `google-analytics`
4. Если запросы есть → установка успешна ✅

### GA4: События для отслеживания

По умолчанию GA4 отслеживает:
- ✅ Просмотры страниц (page_view)
- ✅ Сессии
- ✅ Первые посещения (first_visit)

### GA4: Кастомные события (опционально)

Если вам нужны дополнительные события (отправка формы, клик по кнопке):

**Для разработчика:** Добавить код в `src/components/Contact.js`:

```javascript
// При успешной отправке формы
window.gtag?.('event', 'form_submit', {
  'form_name': 'contact',
  'form_type': 'contact_request'
});

// При клике по телефону
document.addEventListener('click', (e) => {
  if (e.target.href?.includes('tel:')) {
    window.gtag?.('event', 'phone_click', {
      'phone_number': e.target.href
    });
  }
});
```

### GA4: Просмотр отчетов

1. В GA4 перейдите на **Главную страницу**
2. Вы увидите:
   - Количество активных пользователей
   - Просмотры страниц
   - Средняя длительность сессии
   - Показатель отскока
3. Для подробных отчетов → **Отчеты** → выберите нужный

## Яндекс.Метрика

### Шаг 1: Создание счётчика

1. Откройте [Яндекс.Метрику](https://metrica.yandex.ru)
2. Нажмите **"Создать счётчик"**
3. Заполните данные:
   - **Название**: `VECTOR`
   - **Сайт**: `vector-landing-roan.vercel.app`
   - **Часовой пояс**: `Europe/Moscow`
4. Нажмите **"Создать счётчик"**

### Шаг 2: Получение номера счётчика

1. После создания вы получите **ID счётчика** (цифры, например: `123456789`)
2. Скопируйте этот номер

### Шаг 3: Добавление в CMS

1. Откройте CMS вашего сайта (Alt+Shift+C)
2. Найдите поле **Yandex Metrica ID**
3. Вставьте номер счётчика: `123456789`
4. Нажмите **"Сохранить"**

**Альтернатива:** Если нет поля в CMS, отправьте ID разработчику.

### Шаг 4: Проверка установки

1. Откройте ваш сайт
2. В Яндекс.Метрике нажмите **"Проверить установку"**
3. Если все ОК → появится зелёная галочка ✅

### Яндекс.Метрика: События для отслеживания

По умолчанию отслеживаются:
- ✅ Просмотры страниц
- ✅ Переходы между разделами
- ✅ Время проведённое на сайте

### Яндекс.Метрика: Настройка целей

**Цель 1: Отправка формы обратной связи**

1. В Яндекс.Метрике перейдите **Настройки** → **Цели**
2. Нажмите **"Создать цель"**
3. Название: `Отправка формы`
4. Тип: **JavaScript события**
5. Код:
   ```javascript
   contactFormSubmitted
   ```
6. Сохраните

**Цель 2: Клик по телефону**

1. Создайте новую цель
2. Название: `Клик по телефону`
3. Тип: **JavaScript события**
4. Код:
   ```javascript
   phoneClick
   ```
5. Сохраните

### Яндекс.Метрика: Вебвизор

**Что это?** Вебвизор записывает движения пользователя на сайте (мышь, скролл, клики).

**Как включить:**
1. В Яндекс.Метрике перейдите **Инструменты** → **Вебвизор**
2. Нажмите **"Включить вебвизор"**
3. Сохраните изменения

**Важно!** Убедитесь что вы согласованы с [ФЗ-152](https://www.sos-inform.ru/article/FZ-152) (о защите персональных данных).

### Яндекс.Метрика: Просмотр отчетов

1. В левом меню выберите **Отчеты**
2. Основные отчёты:
   - **Обзор** - общая статистика
   - **Посетители** - информация о пользователях
   - **Источники** - откуда приходят люди
   - **Поведение** - какие страницы просматривают
   - **Конверсии** - достижение целей

## Проверка

### Чек-лист установки

- [ ] Google Analytics ID добавлен в CMS (или отправлен разработчику)
- [ ] Яндекс.Метрика ID добавлен в CMS (или отправлен разработчику)
- [ ] При открытии сайта видны запросы к GA и Yandex
- [ ] В GA4 видны "Активные пользователи" в реальном времени
- [ ] В Яндекс.Метрике видны просмотры страниц
- [ ] Цели в Яндекс.Метрике созданы и отслеживают события

### Диагностика проблем

**Проблема:** GA не видит пользователей

**Решение:**
1. Проверьте что ID правильный (G-XXXXXXXXXX)
2. Подождите 24-48 часов (GA обновляет данные медленно)
3. Откройте сайт инкогнито (чтобы избежать Ad Blocker)
4. Проверьте консоль (F12) на ошибки

**Проблема:** Яндекс.Метрика не видит счётчик

**Решение:**
1. Нажмите "Проверить установку" в Яндекс.Метрике
2. Убедитесь что URL правильный
3. Отключите Ad Blocker
4. Подождите 1-2 часа (Яндекс обновляется быстрее GA)
